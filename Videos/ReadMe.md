# AlphaBot2 Run Video

---

# 1. Contents of the Video
This video demonstrates a live run of the Alphabot2 robot navigating autonomously from a start position to a goal using a Q-learning algorithm enhanced with a dynamic model. The robot has no prior knowledge of the environment, encountering obstacles mid-run that it must react to in real time. Relying on its learned policy, the Alphabot2 dynamically adapts its decisions to avoid obstacles and maintain progress toward the goal. This run highlights the practical effectiveness of combining reinforcement learning with dynamic modeling for autonomous robot navigation, demonstrating how the agent successfully handles unexpected conditions to complete the task.
